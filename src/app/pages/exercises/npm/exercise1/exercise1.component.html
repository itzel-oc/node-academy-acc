Exercise 1
Requirements
  • Create a npm project
  • Create a function that creates a file with a log message
  • The function should accept three arguments
  • The first argument should be the name and path of your file; ie. my/path/fileName
  • The second argument should be the Level of the message
  • The third argument should be the message that you want to log
