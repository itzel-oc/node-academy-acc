Exercise 1

Requirements
  • Create a npm project
  • Create an express server that returns an html file
 Setting your npm project
  • Run the following commands
  • Edit the index.js file
> index.js ```javascript const express = require('express'); const path = require('path'); ```
  • Express is an npm module, a routing framework.
  • Path is Node's native utility to work with file and directory paths.

  • Now we init express and set the port we want our app to run in, add this to your index.js file:
> index.js ```javascript let app = express(); let port = 3000; ```
  • Now we will add the code to serve content, add this to your index.js file:
> index.js ```javascript app.use('/', express.static( path.join(__dirname, 'html') )); app.listen(port, console.log('web server listening on port '+port)); ```
  • Run the server
  • Open your browser to this page
